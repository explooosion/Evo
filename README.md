# Evo（ElswordVoice）

This is a ElswordVoice remake project
 + [ElswordVoice](https://github.com/explooosion/elswordvoice)
 + [Download](https://els.robby570.tw/)

## Feature
 + 將軟體面板縮小提供主題自訂切換
 + 取消使用 C# 開發改用 Node.js（Electron）
 + 結合 Firebase 與 FB 粉專訊息系統及時反饋訊息
 + 資源創作共享（提供上傳分享自己的語音或圖片）
 + 軟體資源輕量化
